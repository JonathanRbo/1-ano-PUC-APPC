Objetivo do Sistema

O sistema tem como principais objetivos:
- Registrar solicitações feitas por funcionários.
- Classificar solicitações por categoria e nível de dificuldade.
- Definir prioridade, urgência e impacto das solicitações.
- Direcionar solicitações para equipes responsáveis.
- Controlar o status da solicitação.
- Registrar datas de abertura e atualização. 
- Permitir acompanhamento do progresso das demandas.
